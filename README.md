# OrqueIO SSO Expense

A simple Spring Boot application demonstrating OrqueIO SSO integration with OAuth2/OIDC.

## What's Included

- **Expense Report Process**: A simple BPMN workflow with user tasks assigned to `employees` and `finance` groups
- **SSO Integration**: OAuth2/OIDC configuration with automatic user provisioning and group synchronization

## Prerequisites

- Java 17+
- Spring Boot 3.5.8

## Quick Start

```bash
mvn spring-boot:run
